Create a new repository on the command line
============================================
echo "# htmlassignment" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:souvik-v2/htmlassignment.git
git push -u origin main

=================================================

git branch -M main
